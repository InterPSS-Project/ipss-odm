SCOPE

odm2psat is a Perl script that converts ODM files into PSAT data files.

REQUIREMENTS

Perl and the module XML::Simple must be installed on the system.

USAGE

ODM2PSAT [<OPTIONS>] FILEINPUT [<FILEOUTPUT>]

The options and the file output are optional.

OPTIONS

-h print a this help and exit.
-v verbose output for debugging (currently no effect).
